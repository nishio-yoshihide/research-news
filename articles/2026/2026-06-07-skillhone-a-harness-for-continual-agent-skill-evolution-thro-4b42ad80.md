## 事実要約

SkillHone records diagnoses, revisions, evidence, outcomes, and rejected alternatives so agent skills can improve across sessions without losing the rationale behind prior changes. The authors report gains on GAIA, WebWalkerQA-EN, and seven internal tool-mediated analysis settings; version 2 was posted on June 23.
